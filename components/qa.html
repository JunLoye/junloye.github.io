常见问题解答 / Q&A

💡 提示

这里的回答涵盖了博客功能、技术栈以及如何参与互动的常见问题。

1. 这是一个什么样的博客？

这是 Jun Loye 的个人技术空间。本站基于 GitHub Issues 构建，所有文章内容均同步自 GitHub 仓库。支持 Markdown 渲染、LaTeX 数学公式以及自适应暗色模式。

2. 我可以参与讨论吗？

当然可以。 虽然文章发布权限仅限博主，但欢迎你在每篇文章详情页点击“在 GitHub 评论”按钮。登录 GitHub 后，你可以在对应的 Issue 下方发表看法或进行技术交流。

3. 发现文章有错误怎么办？

如果你在阅读时发现代码错误或文字疏漏，可以登录后点击文章顶部的 “文章有问题” 按钮。这会带你进入 GitHub 的编辑界面，你可以通过提交 Issue 或评论的方式告知我。

4. 音乐播放器支持哪些操作？

右下角的旋转唱片可控制播放与暂停。点击播放器上的文字可展开曲目列表。所有音频文件均托管在 GitHub 仓库中。

5. 搜索功能支持什么？

顶部的搜索框支持对文章标题和摘要进行实时检索，匹配到的关键词会被高亮显示。

还有其他疑问？欢迎通过 GitHub Repository 提交反馈。
