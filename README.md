# Blog | Jun Loye

### ✨ 特性

- **Issue 即博文**：通过 GitHub Issues 进行内容创作，无需本地编译。
- **极简设计**：基于留白艺术与紧凑排版的响应式 UI。
- **动态交互**：根据访问时间自动切换欢迎语（早/午/晚安）。
- **氛围音乐**：内置极简黑胶风格播放器。

### 🚀 快速发布

1. 点击仓库的 **Issues** 选项卡。
2. 使用 **📝 发布新文章** 模板创建新 Issue。
3. 填写标题、上传封面图链接并撰写正文。
4. 提交后，首页将通过 GitHub API 实时同步更新。

### 🛠️ 技术栈

- **Vanilla JS**：原生抓取 GitHub API。
- **CSS3**：Flexbox & Grid 布局，自定义动画。
- **GitHub Actions**：全自动部署。
